# dotfiles

This repo is my first-boot setup for a new computer. It installs the essentials, wires up my shell and editor config, applies a few macOS defaults, and gets the machine feeling less like stock hardware and more like mine.

## Run it

```bash
./setup.sh
```

## Notes

- macOS only.
- Homebrew lives in `~/homebrew`.
- Machine-specific secrets and env vars belong in `~/.profile-env`, not this repo.

If this script finishes and the computer still feels spiritually unarranged, that is now a bug.
